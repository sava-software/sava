# Agent Guidelines

## Documentation

- Documentation should only live in the public-facing hosted docs under `symlinks/docs`.
  When documentation needs to be updated or added (including for behavior or breaking code
  changes), update the hosted docs only.
- Do **not** add or maintain documentation in this repository. In-repo files such as
  `sava-vanity/README.md` should simply link to the relevant hosted docs page rather than
  duplicating its content.
